webappsec-specjp
======================================

W3C Web Application Security Working Group が策定している仕様の日本語訳です。

### 利用上の注意

このリポジトリにある日本語訳はあくまでも参考にすぎず、W3C の勧告は英語版のみが正式なものです。この日本語訳については誤字脱字、翻訳に関する間違いが存在しうることをご了承ください。
